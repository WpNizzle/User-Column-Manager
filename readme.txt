Allows users to add additional columns to the user list and manage custom user data.

== Description ==

User Column Manager is a powerful WordPress plugin that enables users to enhance the user management experience by adding custom columns to the user list. Easily display and manage additional user data, making your user profiles more informative and streamlined.

With User Column Manager, you can:

- Configure custom columns for your user list.
- Input and manage additional user data from user profiles.
- Display your custom column data alongside other user information.

This plugin is designed to simplify user data management and provide you with the flexibility you need to personalize your WordPress admin interface.

== Installation ==

1. Download the plugin ZIP file from the WordPress Plugin Directory.
2. Log in to your WordPress admin dashboard.
3. Go to "Plugins" > "Add New" and click the "Upload Plugin" button.
4. Choose the downloaded ZIP file and click "Install Now."
5. Activate the plugin.

For more detailed installation instructions and usage information, please refer to the [README](https://github.com/WpNizzle/User-Column-Manager) on GitHub.

== Frequently Asked Questions ==

= How do I configure custom columns for the user list? =
After activating the plugin, go to "Users" > "User Column Manager" to configure custom columns. Enter column names separated by commas and click "Save Columns." The custom columns will be added to the user list in the admin.

= Can I edit and manage additional data from user profiles? =
Yes, the plugin allows you to input and manage additional data directly from user profiles. When adding or editing a user, the custom columns will appear for input, and the data entered will be displayed in the user list.
